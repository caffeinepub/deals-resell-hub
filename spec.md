# Specification

## Summary
**Goal:** Revamp the DealDrop frontend visual aesthetic to a refined, premium, and stylish look while retaining the energetic deal-hunting vibe.

**Planned changes:**
- Replace the current bold/loud color palette with deep jewel tones, muted luxe colors, and soft gradients via CSS custom properties in `index.css` and Tailwind tokens in `tailwind.config.js`
- Update typography to use clean, well-spaced, elegant font styles across headings, body text, and price callouts
- Restyle product cards with subtle gradients, refined borders, smooth hover animations, and polished badge design
- Update the header, footer, hero section, and category filter bar to reflect the new aesthetic consistently
- Polish all interactive elements (buttons, inputs, dropdowns) with consistent styling and smooth transitions
- Apply the refined visual language to all admin pages (login, dashboard, listings, orders)
- Restyle countdown timers and discount badges to remain visible but match the new premium aesthetic
- Retain dark background but make it feel premium rather than harsh

**User-visible outcome:** The entire DealDrop site — including all user-facing pages and the admin panel — displays a cohesive, refined, and stylish aesthetic with polished micro-interactions and elegant typography.
